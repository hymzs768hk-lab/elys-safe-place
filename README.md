# Ely's Safe Place

A personal 3D cottage web game built as a warm digital retreat.

## Features

- **Exterior Cottage**: Explore the cozy cottage from the reference image
- **Safe Space**: A bedroom with a diary for writing and reflection
- **Music Room**: Playable instruments and CD collection
- **Art Gallery**: 500 photo frame slots for memories

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to start playing.

## Tech Stack

- React 18
- React Three Fiber + Drei
- Three.js
- React Router
- Zustand for state management

## Data Persistence

All data (diary pages, music, photos) persists via real API endpoints.
